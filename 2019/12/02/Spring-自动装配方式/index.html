Spring-自动装配方式

  • Spring

byName

​ 当使用 byName模式进行自动装配时， Spring会尝i制每每个属性连接到同名的bean。因此， 如果目标 bean具有名为 property的属性并且在 ApplicationContext中定义了名字为property的bean， 那么 property bean将被分配给目标bean的 property 属性。

byType

​ 当使用byType 进行自动装配时， Spring通过在ApplicationContext中自动使用相同类型的bean来尝试连接目标bean模式的每个属性。

构造函数

​ 与 byType 模式在功能上是相同的， 只不过使用的是构造函数而不是setter来执行注 入。 Spring试图匹配构造函数中最大数量的参数。

默认模式

​ Spring将自动在构造函数模式和byType模式之间进行j在择。如果 bean有一个默认的（无参数）构 造函数，那么 Spring使用byType巳模式；否则，就使用构造函数模式。（后续版本中这种方式已经被标记为@Deprecated）。

none

​ 这时默认设置，就是没有使用自动装配，依赖关系需要自己维护。
