  • Java

线程中断

线程中断是一种线程间的学协作方式，设置中断并不能直接终止该线程的执行，而是被中断的线程根据中断状态自行处理。

方法

  1. void interrupt()

    中断线程，这里只是将被调用线程的中断标志置为true，并立即返回。如果被调用线程调用了wait、join或者sleep方法而被阻塞挂起，在另一个线程使用interrupt去中断它时，被调用线程会抛出InterruptException异常返回。

  2. boolean interrupted()

    检测当前线程是否被中断，如果是返回true，否则返回false。注意这里的当前线程，是调用当前线程的中断标志而不是调用interrupted()方法实例的中断标志,如果是主线程去调用子线程的interrupted，判断的是主线程的中断状态。

  3. boolean isInterrupted()

    检测当前线程是否被中断，如果是返回true，同时会清楚中断标志；否则返回false。
