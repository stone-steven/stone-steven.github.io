存储及结构说明

存储

名称 说明
table 存储所有节点数据的数组
slot 哈希槽，即table[i]的位置
bucket 哈希tong，table[i]上的所有元素形成的表或者数的集合

结构

数组+链表+红黑树，当链表长度大于8时转为红黑树。

table定义为Node<K,V>[] table

地址冲突

这使用链地址法来解决哈希冲突。
